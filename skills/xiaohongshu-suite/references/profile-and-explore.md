# Profile And Explore

Use this file when the user wants creator analysis, profile capture, or homepage recommendation review.

## Profile Tasks

Supported profile-style work:

- capture visible profile fields
- compare multiple creators
- inspect posting style
- summarize recurring themes
- compare engagement patterns from visible recent posts

## Preferred Profile Fields

- creator name
- profile url
- bio or intro
- visible follower hints
- visible like-and-collect summary if shown
- recent note titles
- recent note themes
- dominant content style
- posting frequency hints
- engagement hints

If a field is not visible, leave it blank.

## Explore Tasks

Supported explore-style work:

- inspect homepage recommendation flow
- sample hot or recommended notes
- collect visible trends from recommendation cards
- compare recommendation feed versus keyword search feed

## Preferred Explore Fields

- source tab
- card title
- author
- visible engagement
- format hint
- theme or topic hint
- capture time

## Analysis Patterns

### Creator comparison

Use when the user asks:
- which creator is stronger
- which account style fits a niche
- compare competitors

Output:
- Markdown comparison table or CSV

### Explore scan

Use when the user asks:
- what is trending
- what the homepage is currently pushing
- what themes dominate recommended content

Output:
- short trend brief plus supporting table
